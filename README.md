这是一个基于 Vue 3 搭建的个人生活记录网站。在这里，记录日常生活点滴、项目开发过程和学习感悟。  
This is a personal life record website built using Vue 3, where I record daily life, project development, and learning insights.

- 📅 **个人日常记录**：记录每日生活事件。
- 🖼️ **图片展示**：展示个人或团队的照片与活动。
